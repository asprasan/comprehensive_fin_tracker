Finance Dashboard

The Finance Dashboard is a personal tool to track expenses, monitor investments, and manage budgets more effectively. It provides a simple interface for organizing monthly and trip expenses, while also offering insights into long-term financial goals.

The data that you enter here are only available to you locally on the browser. They are not saved anywhere on the internet and no one else can see them. You can export your data to a json file and import it later.

For feedback, issues, and contributions, please checkout the GitHub Project. The existing data that you see in the pages are just sample data and do not reflect anyone's current financial situation.
